Обо мне

«Терпеть не могу отдавать результат, в котором сам не уверен. Если данные выглядят странно — полезу проверять, откуда они взялись, даже если это дольше. Лучше так, чем потом объяснять, почему в отчёте ерунда.»

Последнее, чем занимался: собрал систему, которая следит за конкурентами на рынке счётчиков. Она сама тянет данные из госреестра через API, чистит их, складывает в базу и показывает на дашборде. Писал от начала до конца сам, разбирался по ходу — где-то API отдавал ошибки, где-то цифры не сходились, приходилось придумывать обходные пути. В итоге работает без меня, на автомате.

ИИ использую каждый день, как обычный рабочий инструмент — без религии и без страха. Новое осваиваю быстро, когда это нужно для задачи.
